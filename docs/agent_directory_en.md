    - **Mission:** Strategic decision-making, coordinating all 23 subordinate Agents.

2.  **Core Architect (System Integrity):** 
    - **Mission:** Ensuring source code integrity (Monorepo), scanning for architectural vulnerabilities and system conflicts.
